# Bank Customer Churn Prediction

Predict whether a bank customer will leave the bank using

- Deep Neural Network (TensorFlowKeras)
- XGBoost

## Dataset
Dataset contains customer information such as
- Credit Score
- Age
- Geography
- Balance
- Number of Products
- Active Membership

Target
- Exited (0 = Stay, 1 = Leave)
